# InferaDB Documentation

Welcome to **InferaDB’s official documentation and design repository**.  
This repo hosts all **whitepapers**, **RFCs**, and **technical specifications** that define the evolution of InferaDB’s architecture, policy language, and ecosystem.

---

## **Structure**

| Directory | Description |
|------------|-------------|
| `whitepapers/` | Long-form technical documents and publications. |
| `rfcs/` | Feature and design proposals for review and discussion. |
| `design/` | Deep dives into subsystems and components. |
| `guides/` | Contributor and authoring guides. |
| `diagrams/` | Architecture and flow diagrams (Mermaid or SVG). |
| `templates/` | Document templates for RFCs, whitepapers, and design docs. |

---

## **How to Propose a Change**

1. Fork this repository.  
2. Copy `templates/rfc-template.md` into `/rfcs/`.  
3. Name your file `XXXX-short-title.md` where `XXXX` is the next available number.  
4. Submit a pull request describing your proposal.  
5. Discussion happens in the PR and on the relevant GitHub Discussions thread.
